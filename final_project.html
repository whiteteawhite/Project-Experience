Data cleaning¶

Due to the features Genres, keywords, spoken_languages, production_companies and production_countries columns in the data are all in json format, which is inconvenient to analyze. It is necessary to to convert the json string into a dictionary list form through json.loads. Then, loop is used to select the required data. In this analysis, the corresponding value of the name key is extracted and separated by ",".

json_cols=['cast','crew']
for i in json_cols:
    credits[i]=credits[i].apply(json.loads)
def get_names(x):
    return ','.join([i['name'] for i in x])
credits['cast']=credits['cast'].apply(get_names)
credits['crew']=credits['crew'].apply(get_names)
